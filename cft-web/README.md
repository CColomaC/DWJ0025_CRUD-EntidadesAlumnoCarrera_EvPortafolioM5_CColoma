## Desarrollo Proyecto web dinámico cft-web:
- Un Servlet por cada entidad.
- Organización de los archivos:

  - src/main/java
     - modelo
       - Alumno.java
       - Carrera.java
     - controlador
        - AlumnoController.java
        - CarreraController.java

    - webapp
      - WEB-INF
        - jsp/vista
          - alumno
              - alumno-form.jsp
              - alumno-listado.jsp
          - carrera
              - carrera-form.jsp
              - carrera-listado.jsp
      - index.jsp

## Servlets
- (C)reate  : POST
- (R)ead    : GET
- (U)pdate  : POST
- (D)elete  : GET
